# GospelReads - Next.js Migration

This project is a complete rebuild of the GospelReads website, migrated from Astro to Next.js 15.

## Features

- **Next.js 15 (App Router)**: High-performance React framework.
- **Tailwind CSS 4**: Modern styling with a "New York Times" inspired editorial design.
- **Markdown Content**: Content-driven architecture using `gray-matter`.
- **Static Export**: Optimized for static hosting (Firebase).
- **Responsive Design**: Mobile-first approach for all pages.

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

The static files will be generated in the `out/` directory.
